## 为什么要用markdown

- 写文章的时候不用死磕格式！  
- 易写易读！  
- 优雅！  
- ！  

## 基本语法

- 贴图:

```
![]()
```

- 标题:  

```
# header level 1  
## header level 2  
### header level 3  
#### header level 4  
```

效果: 

# header level 1 
## header level 2
### header level 3
#### header level 4

- 字体

```
*斜体*
**加粗**
_下划线_
~~删除~~
```

效果: 

*斜体*  
**加粗**  
_下划线_  
~~删除~~
